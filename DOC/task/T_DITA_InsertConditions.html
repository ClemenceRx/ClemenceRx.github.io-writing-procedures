Add conditional text into topics

You can insert conditions in topics to filter the content that will appear in the output.

  1. To insert a condition to a topic, right-click the topic in the DITA Maps Manager and select Edit Properties.
    The Edit Topic Properties window opens.
  2. Select the Profiling Tab.
  3. Select the profile that you want to apply to the selected topic. In the following example, the topic will only appear in outputs designed for the technician audience.
  4. In oXygen, select Options > Preferences.
    The Preferences window opens.
  5. Select Editor > Edit Modes > Author > Profiling/Conditional Text.
  6. In the Profiling Condition Sets, select New.
    The Condition Set window opens.
  7. Rename the Condition Set to "Technician Documentation" for example, and select the technician audience.
  8. Select OK.
    The new Profiling Condition Set appears in the list.
  9. Select Apply and OK.
    You can now filter the content that appears in the output files.

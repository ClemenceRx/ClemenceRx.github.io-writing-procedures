Deliver PDF files to PAX Delivery Sharepoint

This procedure describes how to deliver the PAX CHECKPOINT Screening Software documentation other than Relase Notes to the PAX Delivery Sharepoint site.

You need read and write access to the PAX Delivery Sharepoint site. Contact Jean-David Fradette, Jean-David.Fradette@vanderlande.com for support.

  1. In your system, open the directory containing the PDF files generated with oXygen XML Editor.
  2. In your browser, open https://vanderlande.sharepoint.com/sites/T_PAXDeliv/SitePages/Home.aspx.
    The PAX Delivery Sharepoint site opens.
  3. Select Documentation.
  4. Select eVelocity > ReleaseNumber.
    Note: If the Release Number does not exist, create a new folder and name it with the Release Number, for example, 9.21.
    The list of files that are available for the release appears.
  5. Drag and drop the PDF file from your repository to the Sharepoint site window.
    • If you publish an update of a document that already exists, select Add file to confirm that you add this new file as the latest version of the exisitng file.
    • If you publish a new document, the new document appears in the list of documents.

Deliver Release Notes to PAX Delivery Sharepoint

This procedure describes how to deliver the PAX CHECKPOINT Screening Software Relase Notes to the PAX Delivery Sharepoint site.

You need read and write access to the PAX Delivery Sharepoint site. Contact Jean-David Fradette, Jean-David.Fradette@vanderlande.com for support.

  1. In your system, open the directory containing the Release Notes PDF file generated with oXygen XML Editor.
  2. In your browser, open https://vanderlande.sharepoint.com/sites/T_PAXDeliv/SitePages/Home.aspx.
    The PAX Delivery Sharepoint site opens.
  3. Select Downloads.
  4. Select eVelocity > ReleaseNumber.
    Note: If the Release Number does not exist, create a new folder and name it with the Release Number, for example, 9.21.
    The list of files that are available for the release appears.
  5. Drag and drop the PDF file from your repository to the Sharepoint site window.
    • If you publish an update of a document that already exists, select Add file to confirm that you add this new file as the latest version of the exisitng file.
    • If you publish a new document, the new document appears in the list of documents.
